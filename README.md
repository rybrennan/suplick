# Suplick Christmas 2026

The Family Protocol — a festive holiday portal with snow, falling photos, NORAD Ops, Hallmark generator, and more.

## Deploy to Vercel

1. **Push to GitHub** (if not already)

2. **Import in Vercel**
   - Go to [vercel.com](https://vercel.com) → New Project
   - Import your repository
   - **Important**: Set **Root Directory** to `demo`
   - Deploy

3. **Result**: Your site will be live at `https://your-project.vercel.app`
   - Root (`/`) redirects to the Christmas app
   - Direct link: `/christmas`

### Alternative: Deploy from CLI

```bash
cd demo
npx vercel
```

Follow the prompts. The `demo` folder contains everything needed (christmas app + assets).
